# Developer Community App - Privacy Policy

This directory contains the privacy policy for the Developer Community App, hosted on GitHub Pages.

## Files

- **index.html** - The main privacy policy page with professional styling
- **\_config.yml** - Jekyll configuration for GitHub Pages
- **SETUP_INSTRUCTIONS.md** - Step-by-step guide to enable GitHub Pages

## Quick Start

1. Push these files to your GitHub repository
2. Enable GitHub Pages in repository settings (Settings → Pages → Deploy from `/docs` folder)
3. Your privacy policy will be live at: `https://<username>.github.io/<repository>/`

## Customization

To update the privacy policy:

1. Edit `index.html`
2. Update the "Last Updated" date
3. Commit and push changes

## Contact Information

The privacy policy includes placeholder contact information. Please update:

- Email: `privacy@developercommapp.com` (replace with your actual contact email)
- GitHub repository link
